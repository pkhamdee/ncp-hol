# Nutanix Cloud Platform

## Getting Started

ยินดีต้อนรับสู่ Nutanix Cloud Platform (NCP) Part 2 Bootcamp! 👋

แบบฝึกหัด (workbook) นี้ใช้ควบคู่กับเซสชันที่มีผู้สอน ซึ่งจะแสดงวิธีการ build, operate และ govern ด้วย Nutanix cloud operating model โดยเจาะลึกความสามารถดังต่อไปนี้:

- การจับคู่ categories และ policies เข้ากับ entities ที่เกี่ยวข้อง
- การทำความเข้าใจ RBAC และ authorization policies แบบละเอียดสำหรับ Day 2 operations
- การสร้าง Self-Service blueprints สำหรับ deployments ที่สามารถทำซ้ำได้และมีความปลอดภัย
- การใช้ประโยชน์จาก X-Play (cross play) สำหรับ automation ในงานประจำ (routine task)
- การตรวจสอบการใช้งานผ่าน Reporting, Capacity Planning และการระบุ Inefficient VMs

เมื่อสิ้นสุด bootcamp นี้และควบคู่ไปกับ NCP Part1 ผู้เข้าร่วมจะเข้าใจ core concepts และเทคโนโลยีที่ประกอบกันเป็น Nutanix Cloud Platform
