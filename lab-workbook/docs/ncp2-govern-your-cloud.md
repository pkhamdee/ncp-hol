# Operate Your Cloud

Nutanix มอบความสามารถในการ operate ตัว standardized infrastructure ได้อย่างมีประสิทธิภาพผ่าน Intelligent Operations ตัว Nutanix Cloud Manager (NCM) Intelligent Operations เป็นโซลูชันแบบ end-to-end สำหรับ operations, monitoring และ planning ซึ่งผสมผสานแง่มุมต่างๆ ของ administration, reporting และ intelligent automation สำหรับ IT operations

3 เสาหลักที่สำคัญสำหรับ IT organizations ในการ operate ตัว cloud platform ที่ scalable และใช้งานง่าย ได้แก่:

-   **Visibility and Insights to Operate at Scale**
    
    -   การมองเห็น (visibility) แบบ unified ใน infrastructure และ applications, configuration state, metrics และ alerts ทั่วทั้ง on-premises และ hybrid cloud environments ด้วย dashboards และ reporting
-   **Proactive Detection of Performance and Security Issues**
    
    -   เพิ่มประสิทธิภาพ (Streamline) ในการทำ root-cause analysis สำหรับ infrastructure และ application performance โดยใช้ประโยชน์จาก metrics, events และ alerts
-   **Automated Remediation**
    
    -   Automate งานประจำ (routine tasks) และเร่งการแก้ไขปัญหา (issue resolution) เพื่อปรับปรุง productivity

มาเจาะลึก NCM Intelligent Operations กันเถอะ!
