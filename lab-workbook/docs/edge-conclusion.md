# Conclusion

🎉 ยินดีด้วยที่คุณทำ lab นี้สำเร็จ! ขอบคุณแบบฝึกหัดเหล่านี้ 💪 คุณได้เห็นแล้วว่า:

- **NC2** มอบวิธีที่ง่ายในการ migrate **workloads** หรือจัดเตรียม **landing zone** สำหรับ **disaster recovery data center** ทั้ง **Nutanix** และ **cloud administrators** สามารถตั้งค่า **networking** และ **security** เพื่อปกป้อง **workloads** ของตนได้อย่างง่ายดาย
    
- **Nutanix** สามารถให้บริการ **layer 2 stretch** (**Subnet Extension**) เข้าสู่ **public cloud** ทำให้ง่ายสำหรับ **applications** ในการ migrate โดยไม่ต้องทำงานหนักและทดสอบเพื่อเปลี่ยน **IP addresses** โดยที่ **Subnet Extension** ช่วยให้การสื่อสารกลับไปยัง **on-prem** และ **cloud services** ที่ **applications** ซึ่งถูก migrate มาต้องการเรียกใช้งานนั้นเป็นไปได้
    
- **Nutanix Disaster Recovery** มอบวิธีที่ง่ายในการ **failover** เข้าสู่ **public cloud** และอนุญาตให้ **applications** ย้ายกลับไปยัง **private data center** เมื่อระบบกลับมาปกติ **workload mobility** และความยืดหยุ่นในการจัดวางนี้ถูกขับเคลื่อนโดย **Nutanix snapshots** ที่รวดเร็วและคุ้มค่า พร้อมกับเทคโนโลยี **Nutanix network** ที่เป็นนวัตกรรมใหม่
