Conclusion

🎉 Congratulations on completing this lab! Thanks to these exercises 💪 you've seen how:

-   NC2 provides an easy way to migrate workloads or provide a landing zone for a disaster recovery data center. Both Nutanix and cloud administrators can easily set up networking and security to protect their workloads.
    
-   Nutanix can provide layer 2 stretch (Subnet Extension) into the public cloud, making it easy for applications to migrate without having to do a lot of work and testing to change IP addresses. Subnet Extension allows communication back to on-prem and to cloud services that the migrated applications may want to consume.
    
-   Nutanix Disaster Recovery provides an easy way to failover into the public cloud and allows applications to move back to their private data center when healthy. This workload mobility and flexibility in placement is driven by rapid and cost-effective Nutanix snapshots along with innovative Nutanix network technology.
