# Build Your Cloud

standardized infrastructure ที่แข็งแกร่งเป็นจุดเริ่มต้นสำหรับ cloud operating model

**NCM Self-Service** ช่วยจัดการ (orchestrates) การทำ provisioning, scaling และ management ของแอปพลิเคชันในหลาย environments เพื่อทำให้ IT infrastructure ทั้งหมดมีความคล่องตัว (agile) และมุ่งเน้นไปที่แอปพลิเคชันเป็นศูนย์กลาง (application centric) มากขึ้น

3 เสาหลัก (key pillars) ที่สำคัญสำหรับ IT organizations ในการช่วยสร้าง cloud platform ที่ scalable และใช้งานง่าย ได้แก่:

-   **การสร้าง Developer Self-Service Catalog**
    -   Deploy แอปพลิเคชันได้อย่างราบรื่นโดยไม่ต้องจัดการกับความซับซ้อนของการทำ infrastructure management โดยใช้ **Blueprints**
-   **การนำ Catalog ไปอยู่ในจุดที่ Users ต้องการ**
    -   Integrate **Self-Service Marketplace** เข้ากับ workflows เดิมที่มีอยู่และ CI/CD pipelines
-   **การใช้ประโยชน์จาก Infrastructure-as-Code (IaC)**
    -   Automate การทำ Infrastructure deployment และรักษาความสม่ำเสมอ (consistency) ใน environments ต่างๆ โดยใช้ **Self-Service DSL**

มาเริ่มต้นเรียนรู้เพื่อเป็นผู้เชี่ยวชาญในการใช้งาน Nutanix Self-Service กันเถอะ!
