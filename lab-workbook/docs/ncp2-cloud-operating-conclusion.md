# Conclusion

🎉 ขอแสดงความยินดีที่คุณทำ lab นี้สำเร็จ ต่อไปนี้คือบทสรุปของสิ่งที่คุณได้เรียนรู้ในวันนี้เกี่ยวกับ Nutanix Cloud Manager และ cloud operating model

-   **Simplified Day-to-Day Operations**: Nutanix Cloud Platform ช่วยทำให้ daily management ง่ายขึ้น (simplifies) ผ่าน features ต่างๆ เช่น identity and access management ที่ครบถ้วน, projects, categories, และ self-service capabilities สำหรับ workload owners
    
-   **Data-Driven Efficiency**: Nutanix Cloud Manager ใช้ประโยชน์จาก machine learning เพื่อให้ actionable insights ซึ่งช่วยให้เราสามารถตัดสินใจแบบ data-driven เพื่อ optimize ตัว IT operations ของเราได้
    
-   **Resource Optimization is Crucial**: เราได้รับความรู้เชิงปฏิบัติ (practical knowledge) เกี่ยวกับวิธีระบุและจัดการกับ resource inefficiencies ใน virtual machines ของเรา โดยใช้ 4 profiles ที่กำหนดไว้ (Overprovisioned, Constrained, Bully, และ Inactive) สำหรับการทำ right-sizing
    
-   **Automation Simplifies Routine Tasks**: ตอนนี้เราเข้าใจถึงพลังของ low-code/no-code automation ด้วย X-Play และวิธีที่มันสามารถลด manual effort ได้อย่างมากโดยการทำ automating ตัว routine IT tasks ผ่าน playbooks
    
-   **Proactive Capacity Management**: สิ่งสำคัญที่ได้รับ (key takeaway) คือความสำคัญของการทำ proactive capacity forecasting และ planning เพื่อทำความเข้าใจ resource runway ของเรา และเตรียมพร้อมสำหรับ future workload demands ได้อย่างมีประสิทธิภาพ
    
-   **Cost Governance is Essential**: เราตระหนักถึงบทบาทที่สำคัญของการมี cost visibility และ control ทั่วทั้ง infrastructure ของเรา ไม่ว่าจะเป็น on-premises Nutanix หรือใน public cloud ซึ่งต้องขอบคุณ Cost Governance features
    
-   **Nutanix Security Central**: เราได้เรียนรู้วิธีที่ Security Central รวบรวมเอา advanced threat detection, continuous compliance monitoring, และ deep visibility เข้าสู่ hybrid infrastructure—ซึ่งเป็นการเสริมศักยภาพ (empowering) ให้ cloud teams สามารถ manage risk ในเชิงรุก (proactively), enforce ตัว security best practices, และรักษา security posture ที่แข็งแกร่งและสม่ำเสมอทั่วทั้ง environments ต่างๆ
