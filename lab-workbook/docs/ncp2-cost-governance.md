# Cost Governance

ในฐานะ IT organization สิ่งสำคัญคือต้องมี visibility และ control ว่ามีการใช้ resources ไปจำนวนเท่าใด มีค่าใช้จ่าย (costing) เท่าใด และจะสามารถทำ charge back ไปยัง business units ได้อย่างไร NCM Cost Governance มอบ capabilities ทั้งหมดนี้ให้กับคุณและอีกมากมาย สำหรับทั้ง Nutanix on-premises datacenter และ public cloud (AWS, Azure, GCP)

ใน interactive demo ต่อไปนี้ คุณจะได้เรียนรู้วิธีที่ admins สามารถทำความเข้าใจ private cloud costs ของพวกเขา, การทำ map resources เข้ากับ cost centers และ business units, และรับ recommendations เพื่อ optimize ตัว public cloud

Categories จะถูกใช้เพื่อ map resources เข้ากับ Cost Governance สิ่งนี้เชื่อมโยงกลับไปที่ Nutanix Self-Service ซึ่ง blueprints ที่ทำ configured ด้วย categories เฉพาะ และ applications หรือ VMs ใหม่ใดๆ ที่ถูก launched โดย users จะถูก cost governed โดยอัตโนมัติ
