# Takeaways

-   Day 2 และ day-to-day management ด้วย AHV นั้นเป็นเรื่องง่าย
    -   Common tasks ที่ทำใน ESXi ก็สามารถทำได้ใน AHV เช่นกัน
    -   Nutanix Guest Tools สามารถถูก deployed at scale ได้โดยใช้ tools ที่คุ้นเคย
-   Identity and Access Management ที่ครบถ้วนสมบูรณ์ช่วยให้ administrator access และการทำ delegation เป็นไปได้สำหรับทีมขนาดใหญ่
-   การทำ Automation, categories และ policies ใน Nutanix Cloud Manager ช่วยลดภาระของการทำ individual workload management
