# Takeaways

🎉 ขอแสดงความยินดีด้วย! คุณได้เรียนรู้ concepts และ capabilities พื้นฐานของ NKP บางส่วนแล้ว:

-   วิธี expose ตัว applications ใน production โดยใช้ NKP load balancing และ ingress services
    
-   วิธีใช้ NKP Workspaces และ Projects เพื่อทำให้ multi-tenancy เป็นจริง
    
-   วิธี deploy ตัว stateful applications โดยใช้ Nutanix CSI driver ร่วมกับ Nutanix Volumes และ Files
