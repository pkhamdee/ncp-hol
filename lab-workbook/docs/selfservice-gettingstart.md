# Getting Started

ยินดีต้อนรับสู่ Nutanix Cloud Manager Self Service automation and orchestration Bootcamp! workbook นี้ใช้ควบคู่กับ instructor-led session ซึ่งจะแนะนำให้รู้จักกับ Nutanix Self Service และวิธีการนำไปใช้เพื่อจัดหา infrastructure as a service (IaaS) automation, day 2 lifecycle และ operational automation/orchestration ไปจนถึง infrastructure as code (IaC) integration และ extensibility แต่ละส่วนจะมีแบบฝึกหัด (exercise) เพื่อให้คุณได้ทำความรู้จักแบบลงมือปฏิบัติ (hands-on) และฝึกฝน concepts เหล่านี้
