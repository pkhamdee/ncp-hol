# Conclusion

🎉 ขอแสดงความยินดีที่คุณทำ lab นี้สำเร็จ!

การ Migrating ไปยัง Nutanix และโดยเฉพาะอย่างยิ่งไปยัง AHV เป็นเรื่องสำคัญสำหรับ partners และ customers ของเรา มันเป็นสิ่งที่ทุกคนให้ความสนใจเป็นพิเศษ (top of mind) เนื่องจากความเปลี่ยนแปลงล่าสุดในแวดวง virtualization ต่อไปนี้คือบทสรุปของสิ่งที่คุณควรจะได้รับจาก bootcamp นี้:

-   ตอนนี้คุณพร้อมสำหรับการพูดคุยเกี่ยวกับการ migration ไปยัง Nutanix แล้ว ยังมีรายละเอียดอีกมากมายให้ค้นหา พูดคุย และวางแผน นอกเหนือไปจากแค่วิธีการ migrate ตัว VM จาก VMware ไปยัง AHV
    
-   คุณควรมีความเข้าใจเกี่ยวกับ tools และ processes บางส่วนที่คุณสามารถใช้เพื่อดำเนินการ migration โปรดทราบว่า tools ที่เลือกใช้อาจแตกต่างกันไปตามขนาด ความซับซ้อน requirements และ constraints ของโปรเจกต์
    
-   คุณได้เรียนรู้วิธีการใช้งาน Nutanix Move เพื่อทำ VM และ file share migrations แล้ว Move ช่วยให้กระบวนการของการ migrations ประเภทนี้ง่ายขึ้น แต่บาง workloads อาจต้องการ tools เพิ่มเติม
    
-   คุณยังได้เรียนรู้ด้วยว่า Nutanix Cloud Platform สามารถช่วยคุณจัดการ virtual infrastructure แบบ day-to-day ได้อย่างง่ายดายอย่างไร
    

เราหวังว่าสิ่งนี้จะช่วยตอบคำถามที่คุณมีเกี่ยวกับการ migration ไปยัง Nutanix สำหรับพวกเราที่โชคดีพอที่ได้ทำงานในอุตสาหกรรมนี้มาระยะหนึ่ง เราสามารถเห็นได้ว่า migration นี้นั้นง่ายกว่าการทำ physical to virtual migrations ในปีที่ผ่านๆ มามาก

จำไว้ว่า ทีม Nutanix Services ก็มีความเชี่ยวชาญใน migrations ทุกประเภท และสามารถกระโดดเข้ามาให้ความช่วยเหลือในหลายๆ ด้านได้
