# Life Cycle Manager

Nutanix Life Cycle Manager (LCM) ช่วยทำให้ life cycle operations ของ Nutanix IT infrastructure ง่ายขึ้น (simplifies) โดยการรวม (consolidating) การทำ software และ firmware component upgrades ให้อยู่ใน unified control plane เดียวกัน ขณะนี้ LCM สามารถทำ one-click firmware upgrades สำหรับ Nutanix appliances ทั้งหมดได้แล้ว รวมถึง OEM appliances ด้วย

ใน interactive demo ต่อไปนี้ คุณจะได้เรียนรู้วิธีที่ admins สามารถ manage การทำ upgrade ได้ภายในไม่กี่คลิก
