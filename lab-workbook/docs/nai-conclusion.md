# Takeaways

🎉 Congratulations on completing this bootcamp! You have gained familiarity with generative AI applications and the value of the Nutanix Cloud Platform for Enterprise AI.

In this lab, we ran Nutanix Enterprise AI, as well as our generative AI application (Flowise) and a cloud-native vector database (Milvus) on the Nutanix Kubernetes Platform, connecting to our documents on Nutanix Unified Storage.

The Nutanix Cloud Platform provides a full-stack AI solution, powered by Nutanix Enterprise AI, enabling you to run your generative AI applications in your own private datacenter, with your own documents and data stored on Nutanix Unified Storage. This enables you to keep all your data secure and protected, including documents, prompts, and other data, while enabling IT admins to easily manage and control access to the organization's choice of large language models.
