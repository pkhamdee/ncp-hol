# Takeaways

🎉 ยินดีด้วยที่คุณทำ bootcamp นี้เสร็จสมบูรณ์! คุณได้เรียนรู้เกี่ยวกับแอปพลิเคชัน generative AI และคุณค่าของ Nutanix Cloud Platform สำหรับ Enterprise AI แล้ว

ใน lab นี้ เราได้รัน Nutanix Enterprise AI รวมถึงแอปพลิเคชัน generative AI (Flowise) และ vector database แบบ cloud-native (Milvus) บน Nutanix Kubernetes Platform โดยเชื่อมต่อกับเอกสารบน Nutanix Unified Storage

Nutanix Cloud Platform มอบโซลูชัน AI แบบ full-stack ที่ขับเคลื่อนด้วย Nutanix Enterprise AI ช่วยให้คุณรันแอปพลิเคชัน generative AI ใน private datacenter ของตัวเองได้ โดยเก็บเอกสารและข้อมูลบน Nutanix Unified Storage ทำให้ข้อมูลทั้งหมด ไม่ว่าจะเป็นเอกสาร prompt หรือข้อมูลอื่นๆ ปลอดภัยและได้รับการปกป้อง ในขณะที่ IT admin สามารถจัดการและควบคุมการเข้าถึง large language model ที่องค์กรเลือกใช้ได้อย่างง่ายดาย
