# Application: Building a Chatbot and RAG pipeline

**Objective:**  

บทนี้จะช่วยให้คุณมีทักษะพื้นฐานในการทำงานกับแอปพลิเคชัน AI ที่เชื่อมต่อกับ Nutanix Enterprise AI

**What You’ll Do:**

1.  **Create Your First Chatbot**
    
    -   สร้าง chatbot แรกของคุณด้วย Flowise ซึ่งเป็น open source, low-code tool ที่ช่วยให้นักพัฒนาสามารถสร้าง LLM orchestration flow และ AI agent แบบ custom บน LangChain และ/หรือ LlamaIndex โดย LangChain และ LlamaIndex เป็น framework สำหรับลดความซับซ้อนในการพัฒนาแอปพลิเคชัน AI

2.  **Create a RAG pipeline**
    
    -   เพิ่มประสิทธิภาพ chatbot ของคุณด้วย Retrieval Augmented Generation (RAG) ซึ่งช่วยให้ chatbot สร้างคำตอบจากเอกสารที่คุณจัดเก็บไว้
