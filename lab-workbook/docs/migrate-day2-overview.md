# Day 2 Management

ตอนนี้เราได้ทำการย้าย workloads, VMs และ file shares ของเราไปยัง Nutanix เรียบร้อยแล้ว ต่อไปมาสำรวจการทำ Day 2 operations และ management ด้วย Nutanix Prism Central และ AHV กัน

## Key Objectives

ส่วนหนึ่งของส่วนนี้ คุณจะได้:

-   ทำการ Install NGT
-   เรียนรู้เกี่ยวกับ IAM และ user roles
-   เรียนรู้วิธีการทำ automate สำหรับ routine tasks ด้วย playbooks
-   เรียนรู้วิธีการ monitor ตัว environment อย่างต่อเนื่องและเชิงรุก (proactively) ด้วย reports, capacity planning และ inefficient VM detection
