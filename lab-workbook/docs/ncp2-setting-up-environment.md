# Setting Up the Environment

ส่วนนี้เกี่ยวกับการเตรียมความพร้อมสำหรับ hybrid multicloud operations ที่เป็นแบบ automated และมีประสิทธิภาพ สิ่งเหล่านี้ล้วนเป็นองค์ประกอบสำคัญของ cloud operating model

เราจะทำการ configuring องค์ประกอบที่สำคัญดังต่อไปนี้:

1.  Categories - เพื่อจัดระเบียบและอ้างอิงถึง resources ได้อย่างมีประสิทธิภาพ
2.  Role-Based Access Control (RBAC) - เพื่อควบคุมการเข้าถึงของผู้ใช้ (user access)
3.  Policies - เพื่อกำหนดแนวทาง system configuration ที่ใช้กับกลุ่มของ entities ต่างๆ
4.  Projects - เพื่อจัดกลุ่ม resources และ users อย่างเป็นระบบ (logically)

ด้วยการนำ configurations เหล่านี้ไปใช้งาน VMs ของคุณจะดึง settings เหล่านี้ไปใช้แบบ automatically ทันทีที่ถูก deployed บน Nutanix Cloud Platform
